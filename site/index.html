Algorithm Showcase

Rust と egui で実装したアルゴリズム可視化のダッシュボードです。

multi-agent pathfinding

CBS Pathfinding Visualizer

Conflict-Based Search で複数エージェントの衝突なし経路をステップ実行で確認できます。

 constraint solving

WFC Road Visualizer

Wave Function Collapse による道路タイル生成をステップ実行で確認できます。

 information retrieval

Search Index Visualizer

転置インデックス、TF-IDF、コサイン類似度による検索ランキングを表示します。

 distributed consensus

Raft Consensus Visualizer

Raft のリーダー選出・ログ複製・コミットをリアルタイムで可視化します。ノードのクラッシュや遅延を注入して挙動を確認できます。
